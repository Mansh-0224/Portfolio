# Personal Portfolio

A responsive personal portfolio website, currently set up as a clean starter project.

## Project structure

```text
Portfolio/
├── assets/
│   ├── icons/          # SVGs and icon assets
│   └── images/         # Profile, project, and other image assets
├── css/
│   ├── responsive.css  # Breakpoint-specific styles
│   └── style.css       # Base and shared styles
├── js/
│   └── script.js       # Site interactions
├── .gitignore
├── index.html
└── README.md
```

## Planned sections

| Section | Purpose |
| --- | --- |
| Navbar | Provides clear navigation to each main area of the portfolio. |
| Hero | Makes a strong first impression with a brief professional introduction and primary call to action. |
| About | Introduces the person behind the portfolio, including background and professional focus. |
| Skills | Highlights relevant technical and professional capabilities. |
| Projects | Showcases selected work, outcomes, and links to live demos or source code. |
| Contact | Gives visitors simple ways to start a conversation or send an inquiry. |
| Footer | Contains supporting navigation, social links, and copyright information. |

## Getting started

Open `index.html` in a browser. Styling and interactive behavior have intentionally not been added yet.

## GitHub workflow

After creating a GitHub repository, connect and push this project:

```bash
git remote add origin https://github.com/Mansh-0224/Portfolio.git
git branch -M main
git add .
git commit -m "Portfolio Setup"
git push -u origin main
```
